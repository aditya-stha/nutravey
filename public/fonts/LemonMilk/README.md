# LemonMilk Font Files

Place the following files in this directory:

- `LemonMilk.woff2`        (weight: 400, style: normal)
- `LemonMilkMedium.woff2`  (weight: 500, style: normal)
- `LemonMilkLight.woff2`   (weight: 300, style: normal)

The @font-face declarations in globals.css reference these exact filenames.
Once dropped in, the font will load automatically with no code changes needed.
